### 🌐 **TRACK IP / RASTREO IP**
Esta función es el corazón de la aplicación. Permite rastrear direcciones IP remotas y visualizar cada salto hasta el destino. Aquí está la información detallada que se proporciona para cada IP en el recorrido:

<p align="center">
  <img src="https://github.com/TecnoTomer/NetStats_v1.0/blob/main/documents/develop%20gui/intro-frames/track_ip.png" alt="Vista Previa 1" width="600"/>
</p>

- **IP ADDRESS**: La dirección IP.
- **COUNTRY**: País del nodo.
- **REGION**: Región del nodo.
- **CITY**: Ciudad del nodo.
- **COORDINATES**: Coordenadas geográficas.
- **ISP**: Proveedor de servicios de Internet.
- **TIME ZONE**: Zona horaria.
- **LOCAL TIME**: Hora local.
- **DOMAIN**: Dominio asociado.
- **NET SPEED**: Velocidad de red.
- **IDD & AREA CODE**: Código de área y código internacional.
- **ZIP CODE**: Código postal.
- **WEATHER STATION**: Estación meteorológica.
- **MOBILE CARRIERS**: Operadoras móviles.
- **MOBILE COUNTRY CODE**: Código de país móvil.
- **MOBILE NETWORK CODE**: Código de red móvil.
- **ELEVATION**: Elevación sobre el nivel del mar.
- **ASN**: Número de sistema autónomo.
- **IS PROXY**: Indica si es un proxy.
- **PROXY TYPE**: Tipo de proxy.
- **PROXY ASN**: ASN del proxy.
- **SECURITY THREAT**: Amenazas de seguridad.
- **PROXY LAST SEEN**: Última vez que se vio el proxy.
- **PROXY PROVIDER**: Proveedor del proxy.
- **Identificación del Nodo**: Nombre del nodo.

Después de mostrar esta información, se organiza en una vista comprensible y se representa en un mapa, visualizando las rutas y ubicaciones de los saltos.

### 🔍 **SERVICIOS NETSTATS**
Esta función escanea y muestra información sobre los servicios activos en el sistema. Es útil para identificar conexiones y puertos abiertos, ayudando a detectar posibles amenazas de seguridad. Las opciones para cada servicio incluyen:

<p align="center">
  <img src="https://github.com/TecnoTomer/NetStats_v1.0/blob/main/documents/develop%20gui/intro-frames/net_stats_options.png" alt="Vista Previa 1" width="600"/>
</p>

- **Reboot**: Reiniciar el servicio.
- **Details**: Ver detalles del servicio.
- **Kill Task**: Finalizar la tarea del servicio.
- **Properties**: Ver propiedades del servicio.
- **Search Online**: Buscar información del servicio en línea.
- **Show in Folder**: Mostrar el servicio en la carpeta correspondiente.

### ⚙️ **OPTIONS**
En el menú **OPTIONS**, puedes personalizar la aplicación según tus preferencias:
- **Idioma**: Cambia entre inglés y español.
- **Tema**: Alterna entre modo oscuro y claro.
